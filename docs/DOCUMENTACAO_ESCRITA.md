Documentação

Neste trabalho foi desenvolvida uma API utilizando Node.js e Express com o objetivo de aplicar os conceitos estudados em sala de aula. Para o armazenamento dos dados foram utilizados dois tipos de banco de dados: MySQL para o cadastro de usuários e MongoDB para os recursos de carros, motos e marcas de roupa. A autenticação foi implementada com JWT, permitindo o controle de acesso às rotas protegidas da aplicação. Também foram aplicadas medidas de segurança baseadas nas boas práticas da OWASP, como validação de dados, proteção contra ataques de injeção e limitação de requisições. A documentação da API foi gerada com Swagger, facilitando a visualização e os testes dos endpoints. Além disso, foram criados testes de integração utilizando Jest e Supertest para validar o funcionamento dos recursos desenvolvidos. Todo o projeto foi conteinerizado com Docker, permitindo sua execução de forma padronizada em diferentes ambientes. Com isso, foi possível integrar conceitos de desenvolvimento backend, bancos de dados, segurança, testes e conteinerização em uma única aplicação.
